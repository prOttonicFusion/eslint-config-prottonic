# prOttonicFusion's ESLint config

## Usage
1. Install package
    ```
    npm i eslint-config-prottonic
    ```
2. Include it to your ESLint configuration file as:
    ```json
    {
        "extends": "eslint-config-prottonic"
    }
    ```
